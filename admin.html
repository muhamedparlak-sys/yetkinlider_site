Senaryo Yükle

Excel şablonundan senaryo oluşturun veya hazır JSON dosyası yükleyin.

1. Boş Şablon İndir

Excel şablonunu indirin, Google Sheets'e aktarın, doldurun — ardından aşağıdan yükleyin.

Nasıl kullanılır:
1. Excel şablonunu indirin
2. Google Sheets'e aktarın (Dosya → İçe Aktar)
3. "Senaryo" ve "Kararlar" sayfalarını doldurun
4. Dosya → İndir → .xlsx olarak dışa aktarın
5. Aşağıdan yükleyin → JSON oluşturulur → kaydedin

2. Senaryo Yükle

Doldurulmuş Excel veya hazır JSON dosyasını yükleyin.
